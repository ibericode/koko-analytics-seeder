# Koko Analytics Seeder

This plugin registers a new WP CLI command to seed your WordPress database with sample data for the Koko Analytics plugin.

```
wp koko-analytics-seed --years=30 --paths=100 --referrers=100
```

Running this command will generate 100 paths, 100 referrer URL's and 30 years worth of analytics data for [Koko Analytics](https://www.kokoanalytics.com/).

For the reposistory of Koko Analytics itself, please see [ibericode/koko-analytics](https://github.com/ibericode/koko-analytics).
